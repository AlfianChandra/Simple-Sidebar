Code

· Create your button


There are two SimpleSidebar attributes that required to display such as sidebar-target and sidebar-color
Each of them has their own function.

sidebar-target
This will targets the element that will be the sidebar. Insert the ID of the targeted element.

sidebar-color
Sets the sidebar color. You can set any color you want. (RGB, HEX, Color String Such as: red, green, etc)


Append .sidebar-trigger to the button. It allows the button to read those sidebar attributes.

· Create the Sidebar


To simply create the bar, just insert sidebar-control class into the element you want.
And set it's position to arrange it's display direction by adding sidebar-left or sidebar-right.
And give it an ID same as the ID of the button above.
